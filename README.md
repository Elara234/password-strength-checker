# Password Strength Checker

A simple Python tool to check the strength of passwords and provide feedback for improvement. Helps users create secure passwords and understand which aspects need improvement.

## Features
- Checks if a password is Strong, Moderate, or Weak
- Provides actionable suggestions to improve password strength
- Interactive and beginner-friendly CLI application
- Uses Python standard libraries (no external dependencies)

## Tech Stack
- Python
- Regular expressions (re module)
